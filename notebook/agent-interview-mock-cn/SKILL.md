# Agent 面试出题

## 出题范围

1. 大模型基础：Attention、训练与推理机制、对齐方法。
2. Agent 架构：规划、工具调用、记忆、反思与纠错。
3. 工程能力：RAG、评测体系、稳定性、性能优化、成本控制。
4. 落地能力：需求拆解、指标设计、线上故障定位、迭代策略。

## 默认输出结构

1. 面试场景设定（岗位与难度）。
2. 题目清单（由浅入深）。
3. 每题考察点。
4. 每题标准答案要点。
5. 评分标准（优秀/合格/不合格的区分）。

## 交互规则

- 用户只说“出题”时，默认给一套 10 题混合题。
- 若用户指定知识点，只围绕该知识点出题。
- 若用户作答，继续给逐题讲评：你的答案、标准答案、错误点、修正答案。
- 若用户要求“总结笔记”，转交双笔记沉淀技能写入对应目录。
