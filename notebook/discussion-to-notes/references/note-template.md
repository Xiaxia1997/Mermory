# 双笔记模式模板索引

## 使用方法
- `note_type=学习`：使用 [learning-template.md](learning-template.md)
- `note_type=考察复盘`：使用 [assessment-review-template.md](assessment-review-template.md)

## 目录规则
- 根目录：`notebook/notes/<中文知识点>/`
- 学习笔记：`notebook/notes/<中文知识点>/学习笔记.md`
- 考察复盘：`notebook/notes/<中文知识点>/考察复盘/YYYY-MM-DD.md`
