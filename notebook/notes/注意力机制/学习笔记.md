# 注意力机制 学习笔记

## 背景
- 日期：2026-03-08
- 学习目标：建立从 QKV 到梯度与复杂度的完整逻辑脉络
- 当前卡点：softmax 饱和与梯度、`O(n²)` 细节、数值稳定

## 知识逻辑脉络
1. 起点问题：如何在序列中选择并聚合关键信息。
2. 核心概念链路：`Q` 查询、`K` 匹配索引、`V` 内容聚合。
3. 数学机制链路：`QK^T -> /sqrt(dk) -> softmax -> A·V`。
4. 训练与梯度链路：logits 尺度过大导致 softmax 饱和，梯度信号减弱。
5. 工程与复杂度链路：`n×n` 注意力矩阵导致 `O(n²·d)` 时间与 `O(n²)` 显存压力。

## 核心总结
- `sqrt(dk)` 的核心作用是控制 logits 尺度，避免过饱和。
- softmax 梯度要看雅可比矩阵，不是单一标量导数。
- 多头注意力是多子空间互补建模，不是简单加大维度。

## 关键公式与形状
- `Attention(Q,K,V)=softmax(QK^T/sqrt(dk))V`
- `∂p_i/∂z_i = p_i(1-p_i)`，`∂p_i/∂z_j = -p_i p_j`
- `Q(n,dk), K(n,dk), V(n,dv), A(n,n), 输出(n,dv)`

## 面试可背诵版（3-5句）
1. 注意力先做匹配，再做归一化，再对内容加权聚合。  
2. 标准公式是 `softmax(QK^T/sqrt(dk))V`，其中 `sqrt(dk)` 负责尺度稳定。  
3. 不做缩放时，softmax 更易饱和，梯度回传信号会减弱。
